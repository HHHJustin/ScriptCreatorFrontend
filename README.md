# 專案簡介
**LineBot Script Creator (LSC)** 是一個可視化的腳本編輯器，適合用於設計像密室逃脫、互動遊戲等場景，支援：
- 拖拉式節點設計互動流程
- 多分支邏輯設定
- 與 LINE Bot 整合互動流程部署

## LineBot Script Creator 前端操作手冊

這份 README 是支援 **LineBot Script Creator（LSC）** 前端操作的簡易指南，搭配操作手冊，讓你快速上手腳本編輯器的使用方式。
